# E-commerce-de-Bonés
Trabalho de tópicos I do terceiro período de Sistemas de Informação (Universidade Estadual do Tocantins)

# Boné API
API REST desenvolvida em Java com Quarkus para gerenciamento de uma loja de bonés.
O sistema permite cadastro de produtos, controle de estoque, gerenciamento de pedidos e processamento de pagamentos.

A API utiliza JWT para autenticação e autorização, diferenciando permissões entre administradores e clientes.

# Tecnologias utilizadas
1- Java 21+
2- Quarkus
3- Hibernate ORM / Panache
4- JWT (JSON Web Token)
5- PostgreSQL
6- RESTEasy
7- Maven
